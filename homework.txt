1. 
Предварительно нужно сгененрировать .ssh-ключ и добавить его в учетную запись на github.com
Скопировать URL-адрес существующего репозитория или создать новый и скопировать его URL-адрес. 
Добавить репозиторий на компьютер командой:
git clone <URL-адрес репозитория>


2. 
Создать репозиторий в существующем каталоге командой git init
Добавить файлы в индекс командой git add несколько раз, указав индексируемые файлы
Выполнить начальный коммит командой git commit -m "Initial commit"
Создать новый репозиторий на github.com, затем поочередно ввести команды:
git remote add origin <URL-адрес репозитория>
git branch -M main
git push -u origin main


3. 
Изменения с github.com можно получить командами: 
git fetch - без слияния
git pull - со слиянием


4. 
git status - посмотреть список новых или измененных файлов, которые ещё не закоммитены

git log - показать полную историю изменений

git remote -v - просмотреть адреса для чтения и записи, привязанные к репозиторию
