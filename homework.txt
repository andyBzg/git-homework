1. Предварительно нужно сгененрировать .ssh-ключ и добавить его в учетную запись на github.com
Скопировать URL-адрес существующего репозитория или создать новый и скопировать его URL-адрес. 
Добавить репозиторий на компьютер командой:
git clone <URL-адрес репозитория>

2. Создать репозиторий в существующем каталоге командой git init
Добавить файлы в индекс командой git add несколько раз, указав индексируемые файлы
Выполнить начальный коммит командой git commit -m "Initial commit"


3. Изменения с github.com можно получить командами: 
git fetch - без слияния
git pull - со слиянием
