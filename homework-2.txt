1. 
Изменения в файле можно откатить командами

git checkout <имя файла>
git checkout . (с точкой, для отката изменений всех файлов)


2. 
Вернуть файл из индекса можно двумя командами: 

git restore --staged <имя файла>
git reset (более старая команда)


3.
git reset HEAD^
git reset HEAD~1 - цифра указывает на сколько коммитов будем откатываться
